Sovelluksessa on käytetty paljon kurssin aikana läpikäytyjä tekniikoita (mm. navigointi, picker, json-muotoisen tiedon haku). Navigointiin on käytetty Tab Navigation:ia. Lisäksi on käytetty Button:ia ja Alert:ia tietyissä kyselyvisoissa. 
Sovellus arpoo napin painalluksesta uuden kysymyksen(Harry Potter ja Mulptiple Choice Questions) tai laskee pisteet(Personlaity Quizz). Alert ilmoittaa, jos vastaus oli oikein.
UseState:a ja UseEffect:ia on myös käytetty. Taustakuva on tehty ImageBackground:lla.
Harry Potter ja Multiple Choice Questions -visoihin on haettua tietoa json-muodossa netistä.
Monivalinnat on tehty picker:llä.
